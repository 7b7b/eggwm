eggwm
=====

forked from https://code.google.com/p/eggwm/


## Build && Install


### Qt4

```
export QT_SELECT=4
qmake
make
sudo make install
```


### Qt5 for AnthonOS

```
/usr/lib/qt5/bin/qmake
make
sudo make install
```


### Qt5 for ArchLinux

```
export QT_SELECT=5
qmake
make
sudo make install
```
